## Plant Species Gallery

The following 20 flower species were used to train the classification model. Each class contains 250 images for a total dataset of 5,000 images.

| Common Name | Scientific Name | Key Visual Features |
| :--- | :--- | :--- |
| **Anthurium** | *Anthurium andraeanum* | Heart-shaped red spathe with a yellow spike. |
| **Barbeton Daisy** | *Gerbera jamesonii* | Large, colorful ray florets; classic daisy shape. |
| **Bougainvillea** | *Bougainvillea glabra* | Paper-like colorful bracts surrounding tiny flowers. |
| **Californian Poppy** | *Eschscholzia californica* | Cup-shaped orange petals with feathery foliage. |
| **Camellia** | *Camellia japonica* | Layered, symmetrical petals; resembles a rose. |
| **Cape Flower** | *Tecoma capensis* | Tubular orange-red flowers in clusters. |
| **Clematis** | *Clematis viticella* | Large, star-shaped climbing flowers. |
| **Cyclamen** | *Cyclamen persicum* | Upswept petals that look like butterfly wings. |
| **Geranium** | *Pelargonium hortorum* | Clusters of small 5-petaled flowers on long stalks. |
| **Hibiscus** | *Hibiscus rosa-sinensis* | Large trumpet shape with a prominent central stamen. |
| **Lotus** | *Nelumbo nucifera* | Multi-layered aquatic flower with a central seed pod. |
| **Morning Glory** | *Ipomoea purpurea* | Funnel-shaped flowers that open in the sunlight. |
| **Passion Flower** | *Passiflora edulis* | Extremely complex structure with "fringe" filaments. |
